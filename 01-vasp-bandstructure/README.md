# 01 — VASP Band Structure

Compute the electronic band structure and density of states (DOS) of MgO using VASP via an atomate2 workflow.

**Files:**
- `bandstructure.ipynb` — build and submit the `RelaxBandStructureMaker` workflow to FireWorks
- `analyse_bandstructure.ipynb` — retrieve results from the database and plot the band structure and DOS
- `submit_bandstructure.sh` — script to launch jobs on KUMA
